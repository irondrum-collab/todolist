## JWT 만료 처리 검증

자동화 테스트로 실제 토큰 만료를 대기하기 어려워 **코드 검증**으로 대체합니다.

`frontend/src/api/client.ts` 응답 인터셉터:
```typescript
if (error.response?.status === 401 && !error.config?.url?.includes('/auth/login')) {
  window.location.href = '/login';
}
```
- 만료된 토큰으로 API 호출 시 서버가 401 반환
- 인터셉터가 `/login`으로 리다이렉트 처리
- `/auth/login` 엔드포인트는 제외 (잘못된 로그인 정보와 구분)
