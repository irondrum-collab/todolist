## 공통 컴포넌트

### Header (비로그인 상태)

```
+--------------------------------------------------------------------------------+
| TodoList                                          [로그인]  [회원가입]          |
+--------------------------------------------------------------------------------+
```

- 좌측: 앱 로고/서비스명 (클릭 시 `/login` 이동)
- 우측: 로그인, 회원가입 버튼
- 인증이 필요한 페이지 접근 시 라우트 가드가 이 상태에서 `/login`으로 리다이렉트

---

### Header (로그인 상태 — v1.0)

```
+--------------------------------------------------------------------------------+
| TodoList                          홍길동 님  [내 정보]  [로그아웃]              |
+--------------------------------------------------------------------------------+
```

- 좌측: 앱 로고/서비스명 (클릭 시 `/todos` 이동)
- 우측: 현재 로그인된 사용자 이름 표시, 내 정보 버튼, 로그아웃 버튼
- 로그아웃 클릭 시 토큰 삭제 후 `/login`으로 이동

---

### Header (로그인 상태 — v2.0, 테마·언어 추가)

```
+--------------------------------------------------------------------------------+
| TodoList              홍길동 님  [내 정보]  [로그아웃]  [다크/라이트]  [KO|EN] |
+--------------------------------------------------------------------------------+
```

- 좌측: 앱 로고/서비스명
- 우측: 사용자 이름, 내 정보, 로그아웃, 테마 토글 버튼, 언어 선택 버튼
- 테마 토글: 현재 테마에 따라 `[라이트 모드]` 또는 `[다크 모드]` 표시, 클릭 시 즉시 전환
- 언어 선택: `[KO]` / `[EN]` 중 현재 선택된 쪽이 활성화 표시, 클릭 시 즉시 전환

---

### 삭제 확인 다이얼로그 (Modal)

```
          +------------------------------------------+
          |                                          |
          |   정말 삭제하시겠습니까?                 |
          |                                          |
          |   "할 일 제목이 여기 표시됩니다"         |
          |                                          |
          |   이 작업은 되돌릴 수 없습니다.          |
          |                                          |
          |             [취소]    [삭제]             |
          |                                          |
          +------------------------------------------+
```

- 배경에 반투명 오버레이 적용
- 취소 버튼: 다이얼로그 닫기, 목록 유지
- 삭제 버튼: 강조색(빨간색) 적용, 클릭 시 삭제 API 호출 후 목록 갱신
- ESC 키 또는 오버레이 클릭으로도 닫힘

---

### 토스트 알림 (v2.0)

```
                              +---------------------------------+
                              |  저장되었습니다.           [x] |
                              +---------------------------------+

                              +---------------------------------+
                              |  저장에 실패했습니다.      [x] |  <- 오류 시 빨간색
                              +---------------------------------+
```

- 화면 우측 하단 또는 상단에 표시
- 자동으로 3초 후 사라짐, `[x]` 클릭으로 즉시 닫기
- 성공 알림: 초록색 계열, 오류 알림: 빨간색 계열

**인터랙션 설명**

- 수정 버튼 클릭 시 해당 할 일의 데이터가 폼에 미리 채워짐
- 삭제 버튼 클릭 시 삭제 확인 다이얼로그 표시
- 카테고리/상태 필터 변경 시 목록 즉시 갱신
- 할 일 없음 상태: "등록된 할 일이 없습니다" 안내 문구 표시
- 미로그인 접근 시 `/login`으로 리다이렉트 (라우트 가드)

## 상태별 UI 변화

### 할 일 상태 배지 예시

```
+----------------------------------------------------+
| [ ] 할 일 항목 제목 A                              |
|     [시작 전]   기본    2026-06-01 ~ 2026-06-30   |
|                                      [수정] [삭제] |
+----------------------------------------------------+
| [ ] 할 일 항목 제목 B                              |
|     [진행 중]   기본    2026-05-20 ~ 2026-06-10   |
|                                      [수정] [삭제] |
+----------------------------------------------------+
| [x] 할 일 항목 제목 C                              |
|     [완료]      기본    2026-05-01 ~ 2026-05-15   |
|                                      [수정] [삭제] |
+----------------------------------------------------+
| [ ] 할 일 항목 제목 D                              |
|     [기한 초과] 기본    2026-05-01 ~ 2026-05-10   |
|                                      [수정] [삭제] |
+----------------------------------------------------+
| [ ] 날짜 없는 할 일                                |
|     [진행 중]   기본    날짜 미지정                |
|                                      [수정] [삭제] |
+----------------------------------------------------+
```

- `[시작 전]`: 회색 배지, 체크박스 미체크
- `[진행 중]`: 파란색 배지, 체크박스 미체크
- `[완료]`: 초록색 배지, 체크박스 체크 표시 (`[x]`)
- `[기한 초과]`: 빨간색 배지, 체크박스 미체크
- 날짜 미지정 항목은 날짜란에 "날짜 미지정" 표기

### 유효성 오류 표시 예시

**제목 미입력 오류**

```
+---------------------------------------+
| 제목 *                                |
| [                                    ]|  <- 테두리 빨간색 강조
| 제목을 입력해 주세요.                 |  <- 빨간색 인라인 오류 메시지
+---------------------------------------+
```

**날짜 범위 오류**

```
+---------------------------------------+
| 시작일                   종료일       |
| [2026-06-10          ]  [2026-06-01 ] |  <- 종료일 테두리 빨간색
| 종료일자는 시작일자 이후여야 합니다.  |  <- 빨간색 인라인 오류 메시지
+---------------------------------------+
```

**이메일 중복 오류**

```
+---------------------------------------+
| 이메일                                |
| [test@example.com                    ]|  <- 테두리 빨간색
| 이미 사용 중인 이메일입니다.          |  <- 빨간색 인라인 오류 메시지
+---------------------------------------+
```

**현재 비밀번호 불일치 오류**

```
+---------------------------------------+
| 현재 비밀번호                         |
| [                                    ]|  <- 테두리 빨간색
| 현재 비밀번호가 올바르지 않습니다.    |  <- 빨간색 인라인 오류 메시지
+---------------------------------------+
```
